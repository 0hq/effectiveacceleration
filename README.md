# Installation

* Install node.js
* Install npm
* Clone meteor: 
  * cd ~/dev
  * git clone git://github.com/meteor/meteor.git
* Switch to auth branch:
  * cd meteor
  * git checkout auth
* Install meteor:
  * ./install.sh
* Clone MeteorGrid
  * cd ~/dev
  * git clone git://github.com/TroyGoode/MeteorGrid.git 
* Run
  * cd MeteorGrid
  * meteor
